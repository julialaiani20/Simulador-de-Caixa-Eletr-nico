# Simulador-de-Caixa-Eletr-nico
Simulador de Caixa Eletrônico desenvolvido com HTML, CSS e JavaScript. A aplicação permite informar um valor de saque e calcula automaticamente a quantidade de notas necessárias (100, 50, 20, 10, 5 e 2 reais), praticando lógica de programação, manipulação de elementos HTML e eventos de formulário.
