# Simulador de Caixa Eletrônico 💻💰

Projeto desenvolvido com HTML, CSS e JavaScript que simula o funcionamento de um caixa eletrônico. O sistema permite informar um valor para saque e realiza o cálculo automático da quantidade de notas necessárias, utilizando lógica de programação para distribuir valores em cédulas de 100, 50, 20, 10, 5 e 2 reais.

O objetivo do projeto é aplicar conceitos de desenvolvimento web, manipulação do DOM, eventos e criação de interfaces interativas.
